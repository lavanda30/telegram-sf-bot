## 🚀 Запуск за 5 хвилин (безкоштовно)

### Крок 1 — Створіть бота в Telegram
1. Відкрийте [@BotFather](https://t.me/BotFather) у Telegram
2. Надішліть `/newbot`
3. Введіть назву: `Прайс Штори`
4. Введіть username: `prices_shtory_bot` (або будь-який вільний)
5. Скопіюйте **токен** — він виглядає так: `1234567890:ABCdef...`

### Крок 2 — Задеплойте на Railway (безкоштовно)
1. Зайдіть на [railway.app](https://railway.app) → Sign in with GitHub
2. Натисніть **New Project → Deploy from GitHub repo**
3. Завантажте файли `bot.py` та `requirements.txt` у GitHub репозиторій
   (або скористайтесь кнопкою "Deploy from template")
4. У налаштуваннях Railway додайте змінну середовища:
   - **Name:** `BOT_TOKEN`
   - **Value:** ваш токен від BotFather
5. Натисніть Deploy — бот запуститься автоматично!

### Альтернатива — PythonAnywhere (також безкоштовно)
1. Зареєструйтесь на [pythonanywhere.com](https://pythonanywhere.com)
2. Відкрийте **Bash консоль**
3. Виконайте:
```bash
pip install python-telegram-bot==20.7
export BOT_TOKEN="ВАШ_ТОКЕН_ТУТ"
python bot.py
```
